# KPOP Group Order Platform

A platform that helps K-pop fans organize group orders
to complete album sets and obtain member-specific POB photocards.

## Features

- User authentication
- Browse group orders
- Reserve member slots
- Track payment status
- Admin management

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express

Database:
- MySQL
- Prisma
